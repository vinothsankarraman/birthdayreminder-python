# tools-new
this tool is created in Python for Birthday Reminder in Windows.

#Things required
1) Python
2) win10toast package
3) an text file with birthdays(example: mmyy firstname lastname : 0120 James Ford )

#What is does
- reads the content of the text file and checks for todays birthdays.
- pop up windows message if there is any birthday today or no birthdays.
- if there is more one birthday, the pop up gets displayed for all.

#For automatic run
- Schedule the executable file in task scheduler or, (to get executed during a scheduled time)
- Add the executable (shortcut) in the startup of windows (to get executed when system logs in)
